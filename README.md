# RLCraft-Traditional-Chinese-Taiwan
Traditional Chinese (zh_TW) Localization for RLCraft.

RLCraft 的繁體中文翻譯包。
 
 
 
 
====== 使用說明 ======

RLCraft_Traditional_Chinese.zip 直接丟到 resourcepacks 即可使用。

resources.zip 解壓縮後，丟到'PrismLauncher/instances/RLCraft/minecraft'，直接取代原有的 resources 。
