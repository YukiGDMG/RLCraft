# RLCraft-Traditional-Chinese-Taiwan
Traditional Chinese (zh_TW) Localization for RLCraft.

RLCraft 的繁體中文翻譯包。
 
================== 使用說明 ==================

本在地化包分為兩個部分，請務必按照說明使用。

1. 介面與物品漢化 (Resource Pack)
下載 RLCraft_Traditional_Chinese.zip。

將檔案放入 .minecraft/resourcepacks 資料夾中。

在遊戲內的「資源包」選單中將其啟用，並置於最頂層。

2. 任務與系統資源 (Resources Overwrite)
由於部分任務內容儲存在模組設定檔中，需手動替換：

下載 resources.zip 並解壓縮。

找到你的實例目錄 'PrismLauncher/instances/RLCraft/minecraft' 。

================== 問題回報與建議 ==================

如果你在遊戲中發現：
* **翻譯錯誤**：錯字、漏字或語法不通順。
* **顯示問題**：文字重疊、超出視窗或出現亂碼。
* **缺漏補完**：發現仍有部分內容顯示為英文。

請透過 GitHub 的 [Issues](https://github.com/YukiGDMG/RLCraft-Traditional-Chinese-Taiwan/issues) 頁面提交回報。

提交時若能附上「遊戲截圖」與「該物品/任務的英文原名」，將能大幅加快修正速度。感謝你的協助！
